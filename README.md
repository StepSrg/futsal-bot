# Futsal Telegram Bot

Telegram-бот для управления футзальной командой: тренировки, опросы, статистика посещаемости, матчи.

## Функции

- 🏐 **Создание тренировок** с опросом о посещаемости (кнопки ✅/❌/🤔)
- 📊 **Статистика посещаемости** каждого игрока
- 🏆 **Рейтинг** посещаемости по команде
- 👥 **Состав команды** — автоматическая регистрация
- ⚽ **Матчи** — создание, счёт, история
- 📊 **Любые опросы** через /poll

## Быстрый старт

```bash
python3 -m venv venv
source venv/bin/activate
pip install "aiogram<4" aiosqlite
export FUTSAL_BOT_TOKEN="токен_от_@BotFather"
python bot.py
```

## Команды

| Команда | Описание |
|---------|----------|
| `/training ДД.ММ [ЧЧ:ММ]` | Создать тренировку |
| `/stats` | Моя статистика |
| `/top` | Рейтинг команды |
| `/roster` | Состав команды |
| `/match ДД.ММ Соперник` | Записать матч |
| `/score ID СчетНаш СчетИх` | Результат матча |
| `/matches` | История матчей |
| `/poll Вопрос \| Вариант1 \| Вариант2` | Создать опрос |
| `/help` | Помощь |
